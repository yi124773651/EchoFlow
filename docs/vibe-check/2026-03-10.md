## CRITICAL 问题 (2)

### 1. `@Transactional` 包裹 LLM 调用 — 违反 Rule 4

**文件**: `echoflow-application/.../execution/ExecuteTaskUseCase.java:54-90`

```java
@Transactional  // ❌ LLM 调用在事务内
protected Execution planExecution(TaskId taskId) {
    // ...
    var plannedSteps = taskPlanner.planSteps(task.description()); // 远程 I/O!
    // ...
}
```

**Rule 4**: "Never keep a DB transaction open across remote I/O, AI calls, or streaming."

代码中有注释承认了这个问题。

**建议**: 将 LLM 调用移出事务，拆分为 plan → persist 两步：

```java
public void execute(TaskId taskId) {
    var description = loadTaskDescription(taskId);           // 短事务
    var plannedSteps = taskPlanner.planSteps(description);   // 事务外
    var execution = persistPlannedExecution(taskId, plannedSteps); // 短事务
    runExecution(execution);
}
```

### 2. ChatClient 调用无显式 timeout — 违反 Rule 8.6

**文件**: `AiTaskPlanner.java`, `LlmStepExecutor.java`

LLM 调用缺少超时配置。如果 LLM 提供商无响应，线程和数据库连接（与问题 1 叠加）将被无限期持有。

**建议**: 在 `application.yml` 中配置 `spring.ai.openai.chat.options.timeout` 或在 ChatClient 调用中显式设置。

## HIGH 问题 (2)

### 3. 缺少乐观锁 — 违反 Rule 9.4

**文件**: `TaskEntity.java`, `ExecutionEntity.java`, `ExecutionStepEntity.java`

三个 JPA 实体均缺少 `@Version` 字段。异步虚拟线程可能并发更新同一聚合。

**建议**: 为 `TaskEntity` 和 `ExecutionEntity` 添加 `@Version private Long version;`，并在 Flyway 迁移中加列。

### 4. 缺少 fallback/降级策略 — 违反 Rule 8.6

LLM 调用重试 2 次后直接抛异常，无降级方案。

**建议**: 考虑默认响应或标记步骤为 "degraded" 而非直接失败。

## MEDIUM 问题 (2)

### 5. Web 层直接导入 Domain `TaskStatus` — 违反 Rule 4

**文件**: `echoflow-web/.../task/TaskResponse.java`

导入了 `com.echoflow.domain.task.TaskStatus`（域枚举）。Rule 4 允许 Web 层导入 domain ID，但 `TaskStatus` 是业务状态不是 ID。

**建议**: 在 Application 层的 `TaskResult` 中暴露 status，Web 层只用 Application 的返回值。

### 6. 前端任务列表刷新使用 `setTimeout` — 违反 Rule 10.5

**文件**: `echoflow-frontend/src/features/tasks/task-board.tsx:46`

```tsx
setTimeout(loadTasks, 500)
```

Rule 10: "Streaming UIs must use real streaming/SSE; polling is forbidden."

**建议**: 创建任务后直接更新本地 state，或用 SSE 通知列表变更。

## MINOR 问题 (1)

### 7. 日志仅绑定在 Step 上，缺少独立 ExecutionLog 审计表 — Rule 8.3/8.4
